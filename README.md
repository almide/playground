# Almide Playground

Online playground for the [Almide](https://github.com/almide/almide) programming language.

**[Try it live →](https://almide.github.io/playground/)**

## Source

The playground source code lives in [`almide/almide/playground/`](https://github.com/almide/almide/tree/main/playground). This repository is used only as a GitHub Pages hosting target — the `gh-pages` branch is automatically updated when the compiler is pushed to main.
